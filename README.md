# Snippet Stack
This project is deployed at: https://snippet-stack.herokuapp.com

## Dependencies
Python 3.6<br>
SQLite 3.34.0<br>
Flask 1.1.2<br>
gunicorn 20.0.4<br>
Werkzeug==1.0.1<br>


## Installation
To run this project in development mode, use:<br>
pip install -r requirements.txt<br>
python3 snippetApp.py
