# Snippet Stack

This was a group project for CMPT 276 (Intro to Software Engineering)

## Dependencies

Python 3.6\
SQLite 3.34.0\
Flask 1.1.2

## Installation

To run this project in development mode, use:\
`pip install -r requirements.txt`\
`python snippetApp.py`
